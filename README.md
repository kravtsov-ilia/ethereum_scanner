### This utility is designed to find the ethereum wallet which balance has changed the most over the past N blocks

#### Configuration
##### To run the application, you need to have a .env file with following variables
```
API_KEY=your_key
API_ENDPOINT=endpoint (https://eth.getblock.io/mainnet/)
```

#### Run:
`go run .`
or
`go build -o bin/main`
and then `./bin/main`
